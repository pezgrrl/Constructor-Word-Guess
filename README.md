# Constructor-Word-Guess

A Disney villain themed hangman-style game with constructors. This uses node.js, js, npm.

#Requirements:

You will need the npm module inquirer installed to run this activity.

$ npm install inquirer

#Instructions:

enter into gitbash/terminal:

$ node index.js

This will start the game. The object is to guess the correct Disney villain in 10 guesses or less. If you are a winner or loser, the game will prompt you to continue or quit. Play as long as you like.
